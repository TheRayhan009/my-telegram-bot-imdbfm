IMDBFM is a Telegram bot.This project is developed using Python programming language.
IMDBFM is a movie related bot.
Here you can see movie as well as time date and day like sunday or friday.

bot commands : 
1./start
2./help
3./about
4./exit
5./reset
6./again
7./time
8./date
9./day
